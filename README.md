# LeetCode Mastery Guide (NeetCode 150 Model)

This repository contains a curated collection of LeetCode problems organized by topic, following the modern "Easy to Hard" roadmap. Each solution is implemented in **JavaScript** and includes detailed explanations of logic, complexity, and alternative approaches.

### 1. [Arrays & Hashing](./01_Arrays_Hashing)
- **Concepts:** Set, Map, Frequency, Prefix Product.
- **Problems:** `Contains Duplicate` (E), `Two Sum` (E), `Valid Anagram` (E), `Group Anagrams` (M), `Product of Array Except Self` (M).

### 2. [Two Pointers](./02_Two_Pointers)
- **Concepts:** Two pointers, Palindrome, 3Sum, Volume.
- **Problems:** `Valid Palindrome` (E), `Two Sum II` (M), `3Sum` (M), `Container With Most Water` (M), `Trapping Rain Water` (H).

### 3. [Sliding Window](./03_Sliding_Window)
- **Concepts:** Monotonic Deque, Substring frequency, Stock profit.
- **Problems:** `Best Time to Buy Stock` (E), `Longest Substring No Repeating` (M), `Sliding Window Maximum` (H).

### 4. [Stack](./04_Stack)
- **Concepts:** Monotonic Stack, LIFO.
- **Problems:** `Largest Rectangle in Histogram` (H).

### 5. [Binary Search](./05_Binary_Search)
- **Concepts:** Logarithmic search, Rotated arrays, Search space.
- **Problems:** `Binary Search` (E), `Search in Rotated Sorted Array` (M).

### 6. [Linked List](./06_Linked_List)
- **Concepts:** Reversing, Cycle detection, Merging.
- **Problems:** `Reverse Linked List` (E), `Merge Two Sorted Lists` (E), `Linked List Cycle` (E), `Merge K Sorted Lists` (H).

### 7. [Trees](./07_Trees)
- **Concepts:** DFS/BFS, Inverting, Depth calculation.
- **Problems:** `Traversals_Explained.js`, `Invert Binary Tree` (E), `Max Depth of Binary Tree` (E), `Binary Tree Maximum Path Sum` (H).

### 8. [Tries](./08_Tries)
- **Concepts:** Prefix Trees, String indexing.
- **Problems:** `Implement Trie` (M), `Word Search II` (H).

### 10. [Backtracking](./10_Backtracking)
- **Concepts:** Decision Trees, Exhaustive search.
- **Problems:** `Subsets` (M).

### 11. [Graphs](./11_Graphs)
- **Concepts:** Grid DFS/BFS, Sinking nodes, Cycle Detection (Kahn's).
- **Problems:** `Number of Islands` (M), `Course Schedule` (M).

### 12. [1-D Dynamic Programming](./12_1D_DP)
- **Concepts:** Recursion -> Memoization -> Iterative DP.
- **Problems:** `Climbing Stairs` (E).

### 13. [2-D Dynamic Programming](./13_2D_DP)
- **Concepts:** Matrix DP, String distance.
- **Problems:** `Edit Distance` (H).

### 14. [Greedy](./14_Greedy)
- **Concepts:** Local optimum for global goal, Backwards iteration.
- **Problems:** `Jump Game` (M).

### 16. [Intervals](./16_Intervals)
- **Concepts:** Range merging, Sorting.
- **Problems:** `Merge Intervals` (M).

### 17. [Advanced Graphs](./17_Advanced_Graphs)
- **Concepts:** Topological Sort, Alien languages.
- **Problems:** `Alien Dictionary` (H).

### 18. [Math & Geometry](./18_Math_Geometry)
- **Concepts:** Matrix rotation, Spiral traversal.
- **Problems:** `Rotate Image` (M), `Spiral Matrix` (M).
